# DESIGN.md - 奕诺

## 品牌定位
代做服务信任基础设施。为代做交易提供 AI 检测、品类化交付、版权声明三大信任支柱。

## 气质与意象
- 关键词：庄重、可信、仪式感、专业
- 意象：一份精心装裱的契约文书，盖上火漆印章的那一刻 -- 高端律所/公证处的数字化身
- 场景：深胡桃木桌面上摊开的米白色契约纸，琥珀色火漆印章，钢笔书写的签名

## 配色方案
- 主色（墨色）：`#1a1a1a` / `oklch(0.15 0 0)` -- 墨锭研磨后的浓墨
- 辅助色（暖灰）：`#6b6560` / `oklch(0.45 0.01 60)` -- 宣纸经年氧化后的暖灰
- 点缀色（琥珀金）：`#b8860b` / `oklch(0.6 0.12 80)` -- 火漆印章的封蜡金
- 背景色（宣白）：`#faf8f5` / `oklch(0.98 0.003 80)` -- 宣纸的微暖白
- 卡片色（素笺）：`#f5f2ed` / `oklch(0.96 0.005 80)` -- 略带纹理的素笺色
- 成功色（翠绿）：`#2d8a4e` -- 检测通过的印章绿
- 警告色（赭石）：`#c45c26` -- 风险提示的赭石色

## 字体排版
- 标题字体：`Noto Serif SC`（思源宋体）-- 正式、庄重
- 正文字体：`PingFang SC` / `Inter` -- 清晰、现代
- 标题层级：h1=2.5rem/bold, h2=1.75rem/semibold, h3=1.25rem/semibold
- 正文行高：1.75 -- 宽松舒适，像阅读正式文书

## 动效与交互
- 运动性格：克制稳重，像翻动厚重纸张
- 缓动曲线：`cubic-bezier(0.4, 0, 0.2, 1)` -- 缓入缓出
- 过渡时长：200-300ms，不拖沓
- 悬停效果：微妙的底色变化 + 极轻微的 translateY(-1px)

## 设计禁忌
- 不要花哨的多色渐变
- 不要卡通/手绘风格元素
- 不要霓虹色、荧光色
- 不要过度动画和弹跳效果
- 不要圆角过大（保持 6-10px 的克制圆角）
- 不要毛玻璃效果滥用
